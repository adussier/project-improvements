## Second Improvement description


Ajout d'une boutique en jeu permettant d'acheter les 6 upgrade suivantes avec son score:

	- Double tir
	- Triple tir
	- Bouclier
	- Lancer une musique
	- Ralentir les autres pendant une durée limitée
